## Congrats, you now have home server

There are so many possibilities that you have with your server, we can't cover everything. However, we are always open for ideas, questions, recommendations and feedback. In the meantime, you might want to take a look at the [Docker Hub](https://hub.docker.com/) which contains many different (official and unofficial) container images that might be interesting to self-host.

**We hope that you all had fun following this workshop and if you have an idea what to host next, let us know and (if you want to) write the next script :)**
